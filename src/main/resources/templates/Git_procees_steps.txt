Step1: First create a git account, using this url "https://github.com"
Step2: Create repository in git hub and copy that repository  URL
for example "https://github.com/GVenkata-rao/2019.git"
Step3: Open intelliJ IDEA and click on "VCS" --> git --> clone there provide git repository url which u have copied in git hub
and paste git url and select project path then click OK.
